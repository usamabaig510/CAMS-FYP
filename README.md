🎓 College Admission Management System (CAMS)

📌 Overview
The College Admission Management System (CAMS) is a full-stack web application developed to automate and streamline the student admission process. It replaces manual workflows with a digital system for efficient management of applications, verification, and merit-based selection.

🚀 Features
📝 Student Registration System
📂 Document Upload & Verification
📊 Merit List Generation
🔍 Admin Dashboard for Admission Management
📁 Organized Student Data Handling

🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript, Bootstrap
Backend: PHP
Database: MySQL

🎯 Objectives
Automate the admission process
Reduce manual errors and paperwork
Improve transparency in merit-based selection
Provide a user-friendly interface for students and admins
